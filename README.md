# 简介

> 「拜托了快递」 是 **AHU校园快递代取平台**，是AHU软件工程综合实践作品。
> 后端基于Django实现，功能主要分为学生、用户、订单、地址四大模块。
> [前端](https://github.com/TenMoons/Go-Express)基于微信小程序原生语法实现，使用[Lin UI](https://github.com/TaleLin/lin-ui)和自定义组件，采用七牛云对象存储。
